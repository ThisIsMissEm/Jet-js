Download

You can download a prebuilt release of Jet as either a tarball or zipfile

Building from source

You can also build Jet directly from source, using the following:

git clone git://github.com/Miksago/Jet-js.git
cd Jet-js
make

This will give you an uncompressed version of Jet.js in a directory by the name of 'dist'. Make also accepts two other options:

  • make min — Creates a version of Jet.js that has been compressed with Dojo Shrinksafe.
  • make clean — Clean up old files from the 'dist' directory.
