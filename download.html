Download

There's currently no stable releases of Jet.js available for download. However, you can grab the source code from github.

Building from source

Jet comes with a small Makefile and some buildscripts to allow you to easily build a copy of Jet.js. Below is the basic usage: 

git clone git://github.com/Miksago/Jet.js.git
cd Jet.js
make

This will give you an uncompressed version of Jet.js in a directory by the name of 'dist'. Make also accepts two other options:

  • make min — Same as just make, but compressed with Dojo Shrinksafe.
  • make clean — Clean up old files from the 'dist' directory.
