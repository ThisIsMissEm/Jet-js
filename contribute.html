Contributing

Jet.js is an open project, meaning that anyone can contribute to it. There is one catch though, you must sign a copy of the Dojo Foundation's CLA. After that, you can contribute anything you like to Jet.js via patches.

Forking Jet.js

In order to contribute, be sure to fork Jet.js from github.

git clone git://github.com/Miksago/Jet.js.git

Make sure you periodically update your fork off the main repository:

git pull

Merging

Periodically I will visit the forks and see if there is functionality or bugfixes to merge into the project repository.
